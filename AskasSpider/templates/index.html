Askås spider

Askås Spider a web crawler that will go through a website searching for GDPR related errors.

There are three ways to run the spider:

  • Quickscan: The quickest possible way to run the program. Gives the least information.
  • FullScan: The default way to run the program. Gives the most information.
  • AdvancedScan: Customize what areas will be analyzed. Can give as little or as much information as you want.
